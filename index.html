h2>Cerințe de sistem
  • Windows 10 / Linux / macOS
  • Minimum 4GB RAM
  • Conexiune la internet

Pași de instalare

  1. Descarcă arhiva de pe site-ul oficial.
  2. Dezarhivează conținutul într-un folder local.
  3. Rulează fișierul install.exe.
